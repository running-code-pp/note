```
Microsoft 专用

生成 rdtsc 指令，该指令将返回处理器时间戳。 处理器时间戳记录自上次重置以来的时钟周期数。

语法
unsigned __int64 __rdtsc();

返回值
表示滴答计数的 64 位无符号整数。

```

```cpp
#include <stdio.h>
#include <intrin.h>

#pragma intrinsic(__rdtsc)

int main()
{
    unsigned __int64 i;
    i = __rdtsc();
    printf_s("%I64d ticks\n", i);
}
```
