# chroot
全称为change root
改变进程及其子目录的根目录，在linux中默认的根目录是 / 例如 /bash /bin /root /home

chroot newroot [command] ,这里newroot必须是一个完整的文件系统,command 是在新根目录环境下运行的命令 默认是 $SHELL

创建一个最小的环境

mkdir -p /tmp/myroot/{bin,lib64}
cp /bin/bash /tmp/myroot/bin/
cp /lib64/{libc.so.6,ld-linux-x86-64.so.2} /tmp/myroot/lib64/

# 进入 chroot 环境
chroot /tmp/myroot /bin/bash
