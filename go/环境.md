[官网](https://golang.google.cn/)

配置系统变量
- GOROOT设置为go安装目录
- GOPATH设置为go工程目录（可以另外建一个）用户变量中的GOPATH也要设置为一致

# 换源&设置环境变量

# 创建项目
```bash
mkdir test-go
cd test-go
go mod init hello
touch main.go
cat > main.go <<EOF
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
EOF
go run main.go
```
