# 定义
什么是基准测试:
```
基准测试是测试代码性能的方法，可以查看执行速度(延迟)和工作量(吞吐量)
```

## 微观基准测试
类似于单元测试，例如对于某个函数进行基准测试

## 宏观基准测试
类似于集成测试，例如测试某个模块/功能整体

# 测试工具
google benchmark和 catch2

google benchmark可以和googletest很好的配合，而且是头文件库

## 简单使用
```cpp
// 引入googletest
#include <benchmark/benchmark.h>
// 要测试的内容所在的头文件
include "play_game.h"
#include <iostream>

// 迭代对象：benchmark::State &state
static void BENCHMARK_game(benchmark::State &state)
{
    for (auto _ : state)
    {
        for (int i = 1; i <= 100; i++)
        {
            play_game(i, false);
        }
    }
}

BENCHMARK(BENCHMARK_game);

// 运行所有的BENCHMARK的包裹的基准测试函数
BENCHMARK_MAIN();
```
